### 算术运算符: x@y
- `+` 加, 两个对象相加
- `-` 减, 得到负数或一个数减去另一个数
- `*` 乘, 两个数相乘或用作将字符串重复若干次
- `/` 除, x除以y
- `%` 取模, 返回x除以y后的余数
- `**` 幂, 返回x的y次幂
- `//` 取整除向下取接近商的整数

### 比较运算符: x@y 返回True或False
- `==` 等于, 比较两个对象是否相等
- `!=` 不等于, 比较两个对象是否不相等
- `>` 大于, 返回x是否大于y
- `<` 小于, 返回x是否小于y
- `>=` 大于等于, 返回x是否大于等于y
- `<=` 小于等于, 返回x是否小于等于y

### 赋值运算符: x@y
- `=` 赋值运算符, 将y的值(运算结果)赋值给x
- `+=` 加法赋值运算符, 等效于x = x + y, 将y累加到x的值上
- `-=` 减法赋值运算符, 等效于x = x - y, 将x减去y的值
- `*=` 乘法赋值运算符, 等效于x = x * y, 将xy相乘并赋值给x
- `/=` 除法赋值运算符, 等效于x = x / y, 将x除y的商赋值给x
- `%=` 取模赋值运算符, 等效于x = x % y, 将x除y的余数赋值给x
- `**=` 幂赋值运算符, 等效于x = x ** y, 将x的y次方的值赋值给x
- `//=` 取整除赋值运算符, 等效于x = x // y, 将x除y的商向下取整的值赋值给x
- `:=` 海象运算符(赋值表达式), 可在表达式内部为变量赋值, 3.8版本新增
	- 在判断语句中将一些函数/方法返回的对象y赋值给局部变量x, 二者共同作为判断语句的一侧值, 不影响判断结果
	- 详见: [https://docs.python.org/3/whatsnew/3.8.html](https://docs.python.org/3/whatsnew/3.8.html)

### 位运算符: x@y / @x
- `&` 按位与运算符, 如果x和y对应的二进制位都为1, 则该位的结果位1, 否则就为0
- `|` 按位或运算符, x和y对应二进制位有一个为1时, 结果位为1
- `^` 按位异或运算符, x和y对应的二进制位相异时, 结果位为1
- `~` 按位取反运算符, 对x的每个二进制位取反, 即1变0, 0变1
	- 在计算机语言中, 负数采取补码(取反加一)的计算方式, 所以数字~x \== -x-1
- `<<` 左移动运算符, x的二进制位全部左移若干位, 由符号右侧的数决定移动位数, 高位丢弃, 低位补0
- `>>` 右移动运算符, x的二进制位全部右移若干位, 由符号右侧的数决定移动位数, 低位丢弃, 高位补0

### 逻辑运算符: x@y / @x
- `x and y`, 布尔逻辑"与", 如果x为False, 返回x的值, 否则返回y的值
- `x or y`, 布尔"或", 如果x为True, 返回x的值, 否则返回y的值
- `not x`, 布尔"非", 如果x为False, 返回True；如果x为True, 返回False

### 成员运算符: x@y
**其中的序列可以是字符串、列表、元组、字典或集合**
- `in`, 在指定的序列y中找到目标值x返回True, 否则返回False
- `not in`, 在指定的序列y中**未**找到目标值x返回True, 否则返回False

### 身份运算符: x@y
**用于比较两个对象的存储单元**
- `is`, 判断两个标识符是否引用自一个对象, 如果引用自同一个对象则返回True, 否则返回False, 类似id(x) == id(y)
- `is not`, 判断两个标识符是不是引用自不同对象, 如果引用自不同对象则返回True, 否则返回False, 类似id(x) != id(y)
- `id()`函数用于获取对象内存地址

### 运算符优先级:
***当一个表达式中出现多个操作符时, 求值的顺序从高优先级到低优先级, 优先级相同则从左至右***
1. `(expressions)`, `[expressions]`, `{key:value}`, `{expressions}` 括号表达式
2. `x[index]`, `x[index:index]`, `x(arguments...)`, `x.attribute` 读取, 切片, 调用, 属性引用
3. `await x` [[await表达式]]
4. `**` 乘方(指数)
5. `~` 按位取反
6. `-` `+` 一元加减号(正负号)
7. `*` `/` `%` `//` 乘, 除, 求余, 取整除
8. `+` `-` 加法, 减法
9. `<<` `>>` 左移, 右移
10. `&` 按位与(AND)
11. `^` 按位异或(XOR)
12. `|` 按位或(OR)
13. `==` `!=` `<=` `<` `>` `>=` 比较运算符
14. `=` `+=` `-=` `*=` `/=` `%=` `**=`  赋值运算符
15. `is` `is not` 身份运算符
16. `in` `not in` 成员运算符
17. `not` 布尔逻辑非
18. `and` 布尔逻辑与
19. `or` 布尔逻辑或
20. `if--else` 条件表达式
21. `lambda` [[lambda]]表达式
22. `:=` 赋值表达式(海象运算符)
